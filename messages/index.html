Vapor CSS

  • Coveo
  • coveo

Vapor CSS is a collection of UI components used in Coveo's Administration Console. It is widely inspired by Bootstrap 3.0 and borrows some of its components. The project is at its early stages, some components can still have lots of breaking changes.
